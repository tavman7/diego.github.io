# Diego
## Diego Introduction

### What is Diego?

Diego is a popular instruction language. It was created by Tavis Pitt, and released in 2020.

It is used for:

* robot communication,
* thing (of the Internet of Things) (IoT) communication,
* controlling robots,
* controlling things
